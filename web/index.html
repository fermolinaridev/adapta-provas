🤖 Reescrever questões com IA

A IA interpreta cada questão e a reescreve de verdade para o perfil do aluno (não só troca palavras).

🔒 Sua chave fica armazenada localmente (localStorage) no seu navegador. Nunca é enviada para nenhum servidor além do provedor escolhido.
💡 Gemini: se der erro 429 (cota), troque para gemini-1.5-flash que tem limite mais alto (60 req/min na versão gratuita).
💡 Como pegar a chave do Groq (30 segundos, grátis):
  1. Acesse console.groq.com/keys
  2. Faça login (Google/GitHub)
  3. Clique em "Create API Key"
  4. Cole a chave (começa com gsk_) no campo acima
O Groq tem 30 requisições/minuto grátis e é o mais rápido do mercado para Llama 3.3.
→ Chave do Groq (grátis) → Chave do Gemini (grátis) → Chave do OpenRouter
